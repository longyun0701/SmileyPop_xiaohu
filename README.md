# Pop Smileys 打爆笑脸 (by Xiaohu)

## Requirements:
python >= 3.6
pygame >= 1.9.2

## How to play: (Xiaohu writes)

如果您已成功安装pygame和Python，你就可以玩了。
玩法：
    1.把此文件夹里的所有文件全拷到桌面上。
    2.双击SmileyPop3
    3.如果看见“笑脸”，赶快点击它，不然就可能死掉！
    4.特殊按键：
	（1）鼠标点击消除笑脸。
	（2）F4重来
	（3）F12退出（因为游戏是全屏模式，没有退出键）
	（4）F1是什么，我不告诉你！！！
